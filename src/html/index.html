Tutorial Services

Expand the way you learn about Temporal with these browser-based inspection and interaction services. These services extend beyond the standard Temporal Server Web UI to support learning and engagement. Crafted specifically for tutorial use, they enable you to explore developer features in a safe and transparent environment. You can run these locally using 'localhost' (127.0.0.1), or host them on your favorite sites or droplets.

Message Display

A simple display that updates its Markdown-based content using remote API calls. Perfect for exploring scheduled tasks or providing insight into Workflow state.

Action Log

An API-controlled log, with the newest updates always at the top. Create output external to your Workflows to track information you wouldn't normally see in the Web UI or want to query, such as Activity Retries.

Human-in-the-Loop

Explore how you add human intervention for a Workflow to proceed. This microservice provides a friendly button along with a prompt you set. It can't succeed without direct human interaction.

Unreliable Microservice

Tunes a microserve so its reliability ranges from 0.0 (always fails) to 1.0 (always succeeds). Learn to build activity retry policies using an endpoint whose characteristics you control.

Chain Service

Explore how a process moves forward by calling an n-stage chain of microservice activities. You set the chain length and timing with API calls. A Mermaid-based graphic shows the progression of your Workflow through the chain as each Activity enters, retries, and succeeds.

Mantra Service

Similar to the Message Display Service, this service offers a stylized look with the same endpoint behavior. It's to be used with the signals, queries, and schedules samples.
